Start

October 12, 2016

It is time to write a website. First you have to get a text editor.

By default, text editors already are installed on your desktop or laptop. For example:

  • Windows: Notepad
  • Ubuntu: gedit

The text editors above are good, maybe you want to try to get other nice and interesting text editors.

  • Sublime Text (Windows, Linux, macOS)
  • Notepad++ (Windows)
  • Brackets (Windows, Linux, macOS)
  • Atom (Windows, Linux, macOS)

Which text editor is the best? My suggestion is: Choose the text editor you like. You can install some text editors, and try to use, then think what is comfortable for you to write, that is!
