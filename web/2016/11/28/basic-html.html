Basic HTML

November 28, 2016

HTML, the full name is "Hyper Text Markup Language". By writing HTML, we can display contents on browsers. Let's start writing! Open a text editor first! I am used to using "Atom" or "gedit".

The basic HTML form is:



<!DOCTYPE html>
Tell the browsers the HTML version we are using. We use HTML5.

<html></html>
Wrap contents between html tags. The lang attribute has a value named en, it means "English". This html file is written in English.

<head></head>
It includes the webpage's title, metadata, external CSS files or JavaScript files etc.

<body></body>
It includes our main contents, like text, images, videos and other download files and so forth.

Note:

  • Use indentations (soft tab, 2 spaces)
  • We can type <!doctype html>, <!DoCtype html>, <BODy> tags etc. these html tags are case-insensitve.

When finishing the HTML file, save it as "index.html" (the extension ".html" means this is the html file. "index" means the file name. We can name it what we want. We should name the file such as "underscore_name.html" or "hyphen-name.html".)

Finally the simple HTML file is complete. Then click the file, we will see the result on the browser.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>type title</title>
  </head>
  <body>
    <p>type text</p>
  </body>
</html>
